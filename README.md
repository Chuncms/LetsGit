###### 4. 깃 배쉬 맛보기

```$ git init```

```$ git config --global user.name "Your Name"```

```$ git config --global user.email example@abc.com```

```$ git clone <github repo>```
###### 5. 테스트파일 만들기

###### 6. 깃허브에 파일 올리기
1)준비 영역에 파일 올리기

```$ git add *```

```$ git add '파일명'```

2)커밋하기

```$ git commit -m 'descript'```

3) push

```$ git push -u origin master```
